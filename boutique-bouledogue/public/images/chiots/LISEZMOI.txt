Dossier des photos chiots
=========================

Emplacement : boutique-bouledogue/public/images/chiots/

Comment nommer vos fichiers
---------------------------
1) Règle simple : le nom du fichier = le SLUG du chiot + extension.
   Ex. fiche /fr/chiots/lolita  →  fichier : lolita.jpg

2) Si vous préférez un autre nom de fichier : éditez le fichier du projet
   src/data/chiot-covers.ts (CHIOT_COVER_FILENAME_OVERRIDES).

Formats : .jpg, .jpeg, .png, .webp (pour le défaut automatique, utilisez .jpg ou ajoutez une entrée dans chiot-covers.ts avec la bonne extension.)
